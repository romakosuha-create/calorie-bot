# Счётчик калорий — Telegram Bot + Mini App

Полноценный бот для подсчёта калорий с дневником питания и тренировок.
Прообраз: **@ccallories_bot**. Цель — выдавать клиентам, позже монетизировать (пассив).

## 2. Стек

- **Бот:** Python 3.11+, **aiogram 3**
- **API для мини-аппа:** **FastAPI** (async) + валидация Telegram `initData`
- **БД:** PostgreSQL + SQLAlchemy 2 (async) + Alembic (миграции)
- **Мини-апп:** React + Vite + TypeScript + Tailwind + `@twa-dev/sdk`
  - Дизайн — через taste-скиллы (`/design-taste-frontend`, `/imagegen-frontend-mobile`)
- **Штрихкоды:** Open Food Facts API (бесплатно)
- **Планировщик напоминаний:** APScheduler (внутри бота)
- **Деплой:** nginx (статика мини-аппа + reverse proxy на API), systemd (бот + API), Let's Encrypt

## 3. Архитектура

```
Telegram ──┬── Bot (aiogram)         ← команды, напоминания, кнопка запуска Mini App
           │        │
           │        ├── APScheduler   ← ежедневные пуши ("Внесите завтрак")

                    (Telegram WebApp)         │
```

- Бот и API — два процесса, общая БД и общий слой сервисов (`app/services`).
- Авторизация мини-аппа: фронт шлёт `initData` → FastAPI проверяет подпись бота (HMAC) → достаёт `tg_id`.

## 4. Схема БД

- **users** — tg_id, name, tz, пол, дата рождения, рост, вес, уровень активности, цель, цели по ккал/Б/Ж/У, флаги подписки (задел), created_at
- **weight_logs** — user_id, date, weight_kg
- **foods** — name, brand, barcode, kcal_per_100g, protein, fat, carbs, source (off/custom/user)
- **meal_entries** — user_id, date, meal_type (завтрак/обед/ужин/перекус), food_id?, name, grams, ккал, Б, Ж, У, method (manual/barcode/photo), created_at
- **workouts** — user_id, date, type, duration_min, kcal_burned, note
- **water_logs** — user_id, date, ml
- **user_settings** — напоминания (вкл/выкл, время), цель по воде
- **ai_messages** *(позже)* — история чата с нутрициологом

Дневная сводка считается на лету из `meal_entries` (+ расход из `workouts`).

## 6. Формулы

- **BMR (Миффлин–Сан Жеор):**
  - муж: `10*вес + 6.25*рост − 5*возраст + 5`
  - жен: `10*вес + 6.25*рост − 5*возраст − 161`
- **TDEE:** BMR × коэффициент активности (1.2 / 1.375 / 1.55 / 1.725 / 1.9)
- **Цель по ккал:** похудеть −15..20%, держать 0, набрать +10..15%
- **Макросы (по умолчанию):** белок 30% / жир 30% / углеводы 40% (настраивается)
